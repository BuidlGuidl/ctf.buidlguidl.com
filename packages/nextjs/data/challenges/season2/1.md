# #1: The Greeting (Season 2)

Welcome back, players. This is the first challenge for Season 2 of the BuidlGuidl CTF game.

To complete this challenge, you **must first complete Challenge #1 in Season 1**.
Once Season 1 - Challenge #1 is completed (on the same Ethereum address), this Season 2 challenge will automatically be marked as completed as well.

Each flag must be minted to the same Ethereum address for the points to be counted properly.

## Hints

- Make sure you have successfully completed **Season 1 - Challenge #1: The Greeting** with this address.
- You can use the [/debug](/debug) page or [ABI Ninja](https://abi.ninja/) to interact with the contracts.
- Beginners are encouraged to use this [Scaffold-ETH 2 repository](https://github.com/buidlguidl/ctf-argentina/) to solve the challenges. Advanced players may use any means necessary to progress.
